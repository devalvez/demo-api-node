Bem-vindo, a API Demo

Aplicação construida afim de estudar API's em NodeJS e deploy para servidor Heroku

Introdução

Esta API é somente uma aplicação para estudo de deploy com Heroku, e base de dados NoSQL MongoDB, não há funcionalidades disponível para produtos reais e os dados são meramente simbolicos, também não foi levada em consideração questões de segurança.

Usuários

Nesta sessão veremos exemplo de como fazer cadastro de usuários, listagem, consultas e atulização de cadastro.

Listar Usuário

Rota para listar todos os usuários da base de dados.

GET   https://obscure-sierra-39178.herokuapp.com/user

Cadastrar Usuário

Criar novo perfil de usuário

POST   https://obscure-sierra-39178.herokuapp.com/user

Corpo da requisição

parametro Tipo Descrição
name string | required Nome do usuário.
birthday string | required Data de nascimento do usuário.
email string | required E-mail do usuário para para acesso.
socialNetwork 
                            
optional
object {
  facebook: String,
  instagram: String,
  twitter: String,
  linkeding: String
}
URL para o perfil do usuário nas redes sociais

Requisição: 

                  
{
  name: "Jane Doe",
  birthday: "1991-04-27",
  email: "janedoe@domain.com",
  socialNetwork: {
    facebook: "https://facebook.com/username",
    instagram: "https://instagram.com/username",
    twitter: "https://twitter.com/username",
    likedin: "https://linkedin/username"
  }
}
